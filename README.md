# To-Do App

A simple productivity app built with HTML, CSS, and JavaScript.

This project allows users to add, complete, delete, and search tasks. Tasks are saved using localStorage so they remain even after refreshing the browser.

## Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Search/filter tasks
- Floating feedback messages
- Motivational completion messages
- Hover effects and UI improvements
- localStorage persistence

## Technologies Used

- HTML
- CSS
- JavaScript
- localStorage API
